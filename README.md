## 🚀 "NextAuth" - dev camp

<br/>
### 🌟 프로젝트 소개
이 프로젝트는 Next.js를 사용하여 Shacdn/UI, Zod, React Hook Form, 그리고 Resolvers 라이브러리를 활용하여 간단한 회원가입 페이지와 로그인 페이지를 개발하는 것을 목표로 합니다.

<br/>
### 🛠️ 기술 스택
Next.js
Shacdn/UI
Zod
React Hook Form
Resolvers

간단한 웹 애플리케이션 개발, 배포
Next.js, Shacdn/UI, Zod, React Hook Form, Resolvers 사용 연습
